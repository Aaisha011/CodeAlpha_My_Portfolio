Project 1: CALCULATOR

Simple Arithmetic Calculator

Project Description

This project is a basic arithmetic calculator designed to perform standard mathematical operations such as addition, subtraction, multiplication, and division. The calculator is built using HTML, CSS, and JavaScript, ensuring a user-friendly interface and seamless functionality.

Key Features:

  • Addition (+): Adds two or more numbers.
  • Multiplication (*): Multiplies two or more numbers.
  • Division (/): Divides one number by another.

  • Responsive Design: Adjusts to various screen sizes for optimal usability on different devices.

Technologies Used:

  • HTML: Structures the calculator interface.
  • CSS: Styles the calculator for an appealing and intuitive user experience.
  • JavaScript: Implements the functionality for arithmetic operations and userinteractions.

Project 2: Image Gallery

Image Gallery

Project Description

The Image Gallery project is a web-based application that allows users to browse, view, and interact with a collection of images. This gallery is built using HTML for the structure, CSS for styling and layout, and JavaScript for interactivity. Instead of using CSS Grid, Flexbox is utilized to create a responsive and flexible layout for the gallery.

Key Features:

  • Flexbox Layout: The gallery uses Flexbox to arrange images in a responsive, fluid layout. Flexbox allows images to wrap into new lines as the viewport size changes, providing a consistent and adaptive display across devices.
  • Navigation Controls: Users can navigate through the images using next and previous buttons within the lightbox.
  • Responsive Design: The Flexbox layout ensures that the gallery is responsive, providing an optimal viewing experience on mobile devices, tablets, and desktops.

Technologies Used:

  • HTML: Structures the gallery, including image elements and containers.
  • CSS: Styles the gallery, with Flexbox used for the layout. CSS also handles responsiveness.
  • JavaScript: Adds interactivity, such as opening the lightbox, navigating images, and implementing lazy loading.
